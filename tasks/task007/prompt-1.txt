在当前 React 项目中实现「表单构建器」页面，所有相关代码放在 src/pages/form-builder/ 下。该页面是首页任务卡片「表单构建器」点击后的二级页面，用 localStorage 做数据持久化。

1. 左侧是字段组件面板，列出所有可用的字段类型：单行文本框、多行文本框、下拉选择框、单选框组、多选框组、日期选择器、数字输入框、开关。从左侧拖拽字段到中间画布区域即可添加一个对应的表单字段。
2. 点击画布中某个字段后，右侧显示属性编辑面板，可配置的内容取决于字段类型，至少包括字段标签、是否必填、占位符文字；下拉和单选/多选框还需要能编辑选项列表；数字输入框需要可设最大最小值。
3. 画布中的字段支持拖拽调整排列顺序，也支持删除。顶部放一个预览按钮，切换到预览模式后画布变成一个可交互的真实表单。
4. 提供导出功能：将当前画布中的表单结构导出为 JSON Schema 格式并触发下载。
5. 构建状态持久化到 localStorage，刷新后不丢失。
6. 在 src/pages/__tests__/form-builder/ 下编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
7. 在 src/App.jsx 中注册该页面的路由，确保从首页点击卡片能正确跳转。同时做好响应式布局，在 375px 和 1440px 宽度下均可正常使用。
