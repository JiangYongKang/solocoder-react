在当前 React 项目中实现「主题定制系统」页面，所有相关代码放在 src/pages/theme-editor/ 下。该页面是首页任务卡片「主题定制系统」点击后的二级页面，用 localStorage 做配置持久化。

1. 页面分左右两个功能区。左侧是控制面板，包含三组配置：主题模式切换（亮色/暗色一键切换）、颜色编辑器（用 CSS 变量驱动，列出主色、背景色、文字色、边框色、强调色、成功色、警告色、错误色等关键颜色，每个颜色旁放颜色拾取器，修改后右侧预览区即时更新）、字体与间距调节（字体大小 12-24px 可调，行高和段落间距也可调）。
2. 右侧是预览区，展示按钮、卡片、输入框、表格、提示信息等典型 UI 元素在当前主题配置下的实际效果，样式完全由左侧控件的值驱动。
3. 支持导出当前全部主题配置（模式、颜色、字体、间距）为 JSON 文件并触发下载。
4. 支持导入 JSON 配置文件，解析后应用其中的所有配置项到当前主题。
5. 主题配置持久化到 localStorage，刷新后恢复上次配置。
6. 在 src/pages/__tests__/theme-editor/ 下编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
7. 在 src/App.jsx 中注册该页面的路由，确保从首页点击卡片能正确跳转。同时做好响应式布局，在 375px 和 1440px 宽度下均可正常使用。
