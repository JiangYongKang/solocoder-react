在当前 React 项目中实现「富文本编辑器」页面，所有相关代码放在 src/pages/rich-editor/ 下。该页面是首页任务卡片「富文本编辑器」点击后的二级页面，用 localStorage 做自动保存。

1. 编辑器分左右两栏：左侧是编辑区，右侧是 Markdown 实时预览面板。编辑区支持键盘输入，工具栏提供加粗、斜体、下划线、删除线、标题（H1-H3）、引用、有序列表、无序列表、代码块等格式按钮。
2. 支持插入图片（通过粘贴 URL 或本地 FileReader 预览）和超链接。
3. 实现操作栈，支持撤销和重做。
4. 编辑区内容自动保存到 localStorage，刷新后不丢。提供两个导出按钮：导出为 HTML 和导出为 Markdown 文件。
5. 在 src/pages/__tests__/rich-editor/ 下编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
6. 在 src/App.jsx 中注册该页面的路由，确保从首页点击卡片能正确跳转。同时做好响应式布局，在 375px 和 1440px 宽度下均可正常使用。
