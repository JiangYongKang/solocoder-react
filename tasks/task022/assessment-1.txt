2. 模型在 diffUtils.js 第336-342行定义了 escapeHtml 函数，但在整个 text-diff 包内的 index.jsx 和所有其他文件中均无任何调用。该函数是死代码，产物不干净。根本原因是模型预设了文本转义需求但实际 React JSX 已自带 XSS 防护无需此函数，写完后又未做 dead code 清理。正确做法是在代码完成后进行全局 grep 检查，移除未使用的导出。
