在当前项目中实现文本差异对比工具，该页面是首页任务卡片"文本差异对比工具"点击后的二级页面。

这个页面是一个纯粹的文本比对工具，用户输入两段文本后直观看到差异。

1. 主区域采用左右分栏的对比视图，左侧是原始文本，右侧是修改后的文本，两侧同步滚动。差异行用绿色（新增）、红色（删除）、黄色（修改）底色标记。
2. 除了逐行差异外，同一行内的具体不同字符也要用不同深浅的背景色高亮，让用户能精确看到每个字符级别的变化。行号始终对齐显示。
3. 在左右对比视图和统一 diff 视图之间做一个切换按钮。统一视图类似 Git diff 的单栏模式，用 + 和 - 前缀表示增删行，并带行号。
4. 文本输入支持三种方式：直接在左右两侧的文本编辑框中粘贴内容、上传本地 .txt 或 .md 文件读取内容、从剪贴板一键粘贴。
5. 右侧做一个差异块导航面板，列出所有变更块（显示变更位置和变更类型），点击任意一块可以快速滚动跳转到对应位置。
6. 代码实现需要放在 src/pages/text-diff/ 下，测试代码放在 src/pages/__tests__/text-diff/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
7. 在 src/App.jsx 中注册该页面的路由，确保从首页点击卡片能正确跳转。所有页面需做响应式布局，在 375px 和 1440px 宽度下均可正常使用。
