在当前项目中实现思维导图编辑器，该页面是首页任务卡片"思维导图编辑器"点击后的二级页面，使用 localStorage 做数据持久化。

用户在这个页面里可以创建和编辑树形层级的思维导图，所有渲染基于 Canvas 或 SVG 自绘。

1. 画布支持无限层级的节点展开，从中心根节点向外发散。点击节点旁的 +/- 按钮可以展开或折叠子节点。节点之间用直线或折线连接。
2. 支持在任意节点上添加子节点、编辑节点文字、删除节点。节点之间支持拖拽排序——拖拽一个节点到另一个节点上可以改变其父子关系，或调整同级顺序。
3. 节点可以自定义颜色和图标。右键点击节点弹出菜单，提供颜色选择器和一组预设图标（如旗帜、星星、灯泡、感叹号等），选中后实时更新节点样式。
4. 画布支持鼠标滚轮缩放和拖拽平移，缩放范围在 30% 到 200% 之间。画布右下角显示当前缩放比例，并提供「适应画布」按钮一键居中。
5. 支持快捷键操作：Tab 添加子节点、Enter 添加同级节点、Delete 删除节点、F2 编辑节点文字、方向键在同级节点间切换选中。
6. 顶部工具栏提供 JSON 导入导出功能——导出当前脑图为 JSON 文件下载，也支持从本地 JSON 文件导入还原。同时提供「导出为 PNG」按钮，将当前画布可见区域导出为图片下载。
7. 代码实现需要放在 src/pages/mind-map/ 下，测试代码放在 src/pages/__tests__/mind-map/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
8. 在 src/App.jsx 中注册该页面的路由，确保从首页点击卡片能正确跳转。
