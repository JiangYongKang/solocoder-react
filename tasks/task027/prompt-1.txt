在当前项目中实现代码片段管理器，该页面是首页任务卡片"代码片段管理器"点击后的二级页面，使用 localStorage 做数据持久化。

这个页面是开发者的个人代码片段收藏夹，支持管理和快速查找自己的代码片段。

1. 代码编辑区使用带行号的代码编辑器（textarea 模拟即可，需等宽字体），输入代码后自动根据语言标签进行语法高亮渲染。语法高亮做基本的 JavaScript、TypeScript、Python、CSS、HTML 五类支持即可——关键词、字符串、注释用不同颜色显示。
2. 每条代码片段包含标题、语言标签、代码内容、Markdown 备注四个字段。创建片段后，语言标签作为分类维度，顶部提供按语言标签筛选的标签栏，点击标签过滤显示对应语言的片段。
3. 全局搜索框支持按标题和代码内容搜索，搜索词在结果列表中高亮。搜索结果可以按创建时间或按名称排序。
4. 支持收藏片段（星标），收藏的片段优先显示在列表顶部。每个片段卡片提供一键复制代码到剪贴板的功能，复制后显示「已复制」的短暂提示。
5. 列表支持卡片视图和紧凑列表视图两种展示模式切换。卡片视图显示标题、语言标签、代码前几行的预览；列表视图只显示标题和语言标签。
6. 顶部工具栏提供 JSON 导入导出——导出所有代码片段为 JSON 文件下载，支持从 JSON 文件导入片段（可选择覆盖或合并）。
7. 代码实现需要放在 src/pages/snippets/ 下，测试代码放在 src/pages/__tests__/snippets/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
8. 在 src/App.jsx 中注册该页面的路由，确保从首页点击卡片能正确跳转。所有页面需做响应式布局，在 375px 和 1440px 宽度下均可正常使用。
