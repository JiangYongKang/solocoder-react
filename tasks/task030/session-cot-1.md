## ✨ 功能实现

1. **SVG 环形进度倒计时** - 使用 SVG 绘制，三种阶段（工作25分钟/短休5分钟/长休15分钟）不同颜色显示
2. **计时控制** - 开始/暂停/停止，暂停后继续从剩余时间计时，停止重置当前阶段
3. **浏览器标题更新** - 实时显示如 `24:59 - 工作中`
4. **自动阶段切换** - 每完成 4 个工作番茄自动进入长休，可配置 2-6 的间隔
5. **自定义时长** - 工作/短休/长休时长均可自定义，带输入验证
6. **任务绑定** - 输入任务名称，完成后自动记录（时间戳 + 任务名 + 时长）
7. **统计图表** - 近 7 天/30 天的番茄数趋势，柱状图/折线图切换，显示总番茄数、总专注时长、日均番茄数
8. **桌面通知** - 各阶段结束发送 Notification API 提醒，首次使用自动请求权限
9. **提示音** - 使用 Web Audio API 生成蜂鸣声
10. **白噪音** - 5 种选择（雨声、咖啡馆、森林、白噪音、静音），选中后显示播放状态
11. **localStorage 持久化** - 设置和完成记录自动保存
12. **响应式布局** - 375px 到 1440px 宽度下均正常显示
