你实现的 API 调试工具页面存在一些问题，需要进行修复：

4. 修复完成后启动 pnpm run dev，手动验证从首页点击"API 调试工具"任务卡片能正常进入页面。
5. 重新运行 pnpm run test，需要让所有测试通过。
