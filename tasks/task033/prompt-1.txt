在当前项目中实现协作白板，该页面是首页任务卡片"协作白板"点击后的二级页面，所有绘图功能基于 Canvas 自绘。

这个页面是一个自由绘制的白板工具，用户可以在无限画布上绘制图形、书写文字和进行简单的创作。

1. 左侧工具栏提供绘制工具选择：画笔（自由绘制线条）、矩形、圆形、直线、文本、橡皮擦。选中工具后，在画布上按住鼠标拖拽即可绘制对应图形。画笔支持颜色选择器和线宽调节（1-10px 滑块）。橡皮擦以圆形区域擦除画布内容，也支持调节大小。
2. 矩形和圆形绘制时，拖拽过程中实时显示形状预览边框。松开鼠标后形状固化，形状内部用半透明或透明填充，边框颜色和线宽与画笔共用颜色和粗细设置。直线工具支持按住 Shift 锁定水平/垂直/45 度角。
3. 文本工具点击画布后弹出输入光标，用户直接输入文字，支持字体大小和颜色设置。已绘制的文本再次点击可重新编辑。
4. 撤销和重做：顶部工具栏提供撤销和重做按钮，支持多步撤销和重做——撤销刚画的图形或文字可恢复，重做可重新应用。同时支持 Ctrl+Z 撤销、Ctrl+Y 重做快捷键。
5. 画布操作：支持鼠标滚轮缩放（50%-300%）和鼠标中键或按住空格键拖拽平移。左下角显示当前缩放比例，并提供「适应窗口」按钮一键恢复。
6. 文件操作：顶部工具栏提供「导出 PNG」按钮，将当前画布可见区域导出为 PNG 图片下载。提供 JSON 导入导出功能，将画布上所有图形数据（类型、位置、颜色、大小、文字内容等）序列化为 JSON 文件下载，也支持从 JSON 文件导入恢复画布。
7. 代码实现需要放在 src/pages/whiteboard/ 下，测试代码放在 src/pages/__tests__/whiteboard/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
