在当前项目中实现数据库表设计器，该页面是首页任务卡片"数据库表设计器"点击后的二级页面，使用 localStorage 保存和恢复设计。

这个页面是一个可视化的数据库表结构设计工具，用户通过拖拽和连线来设计表与表之间的关系。

1. 画布区域为主体操作区，支持无限缩放和平移。左侧提供工具栏，包含「新建表」按钮和表模板。点击「新建表」或拖拽表模板到画布上创建一个新表。新表默认包含一个 id 主键字段，表头显示表名（默认可编辑），表体列出所有字段。
2. 字段编辑：每个表内支持添加、编辑、删除字段。每个字段包含字段名、字段类型（下拉选择：INT、VARCHAR、TEXT、BOOLEAN、DATE、FLOAT、DECIMAL）、是否主键（勾选框）、是否允许 NULL（勾选框）、默认值输入。主键字段在字段名左侧显示钥匙图标，不可勾选允许 NULL。支持拖拽字段行调整字段顺序。
3. 表之间可以建立外键关系：从一个表的字段拖拽连线到另一个表的主键字段，连线自动吸附到两端。连线用贝塞尔曲线或折线绘制，箭头指向被引用表。右键点击连线可以删除外键关系。
4. 画布操作：支持鼠标滚轮缩放（50%-200%）和拖拽平移。表可以在画布上自由拖拽移动位置。画布上的连线跟随表位置自动重新布局。提供「自动布局」按钮，一键按层级重新排列表位置。
5. 导出 DDL：点击「导出 DDL」按钮，根据当前所有表和字段定义生成标准 SQL DDL 语句（CREATE TABLE），包含字段类型、主键、NOT NULL 约束、FOREIGN KEY 引用。DDL 以文本形式展示并可复制。支持单表导出和全部导出两种模式。
6. 保存与恢复：设计数据自动保存到 localStorage。提供 JSON 导出，将当前所有表结构、字段定义、连线关系保存为 JSON 文件下载。支持从 JSON 文件导入恢复设计。
7. 代码实现需要放在 src/pages/db-designer/ 下，测试代码放在 src/pages/__tests__/db-designer/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
