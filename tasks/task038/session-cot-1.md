## ✅ 已实现的功能

1. **画布操作**：无限缩放（50%-200% 滚轮）、拖拽平移、点状网格背景
2. **左侧工具栏**：「新建表」按钮 + 3 个预置模板（用户表/商品表/订单表），支持点击或拖放创建
3. **表结构**：新表默认带 id 主键字段，表名可直接编辑，支持自由拖拽移动
4. **字段编辑**：7 种类型（INT/VARCHAR/TEXT/BOOLEAN/DATE/FLOAT/DECIMAL）、主键勾选（显示🔑图标且强制 NOT NULL）、NULL 勾选、默认值输入、拖拽排序、添加/删除
5. **外键关系**：从字段圆点锚点拖拽连线到另一表主键字段，贝塞尔曲线 + 箭头，右键删除
6. **自动布局**：基于 Kahn 拓扑排序算法按外键依赖分层排列
7. **DDL 导出**：单表/全表两种模式，按依赖拓扑排序输出，含 PRIMARY KEY、FOREIGN KEY REFERENCES，可复制
8. **持久化**：自动保存到 localStorage；JSON 导出下载/导入恢复，带完整 schema 校验
