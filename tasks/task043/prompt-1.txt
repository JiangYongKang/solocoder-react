在当前项目中实现多语言国际化管理，该页面是首页任务卡片"多语言国际化管理"点击后的二级页面，使用 localStorage 做数据持久化。

这个页面是一个管理应用多语言翻译键值对的工具，支持多语言并行编辑、导入导出和翻译覆盖率统计。

1. 键值对表格编辑：以可编辑表格形式展示所有翻译键值对。第一列为翻译 key（支持嵌套 key，如 user.profile.name，用点号分隔表示层级），后续每列为一个语言的翻译值。表格行支持添加、删除和修改 key 和对应值。key 不可重复，值为空时高亮标记。
2. 嵌套 key 树形展开：支持将嵌套 key 按层级展开为树形视图。一级 key（如 user）作为父节点可展开收起，展开后显示子级 key（如 user.profile.name 仅显示 name，前缀折叠）。提供「树形」和「平铺」两种视图模式切换按钮。
3. 并行对比视图：提供一个对比视图，将所有语言的翻译值并排展示。选中某个 key 后，对比面板中每列显示一种语言对该 key 的翻译，不同语言之间翻译一致时标绿，存在差异时标黄，某语言未翻译时标红。
4. 新增语言与 JSON 导入导出：支持动态新增语言——点击「添加语言」按钮输入语言名称和语言代码（如 zh-CN、en-US），新增的语言列自动加入表格。支持 JSON 导入：上传一个 JSON 文件（格式为 { "key": { "zh-CN": "...", "en-US": "..." } } ），文件内容合并到现有数据中（同 key 覆盖，新 key 追加）。支持 JSON 导出：将当前所有翻译数据导出为 JSON 文件下载。
5. 未翻译 key 筛选：提供筛选功能——「显示全部」「仅显示未完全翻译」「仅显示完全翻译」。未完全翻译指任意一种语言该 key 的翻译值为空。筛选下拉框切换后表格实时过滤。
6. 翻译覆盖率统计：在页面顶部展示翻译覆盖率统计面板。每个语言显示一个进度环或进度条，百分比为该语言已翻译的 key 数 / 总 key 数。同时显示总 key 数和各语言已翻译 key 数。统计数值随表格数据变化实时更新。
7. 代码实现需要放在 src/pages/i18n-manager/ 下，测试代码放在 src/pages/__tests__/i18n-manager/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
