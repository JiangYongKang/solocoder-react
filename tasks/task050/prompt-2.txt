你实现的几何画板还存在一些问题，需要进行修复：

1. 线段绘制在 Canvas 和 SVG 导出中都缺少两端点的坐标标注文字。线段目前只在中点显示了长度标注和端点圆点，但没有在各端点处显示坐标值。提示词要求线段显示两端点的坐标标注和长度标注。

2. 坐标轴箭头的位置使用了硬编码的 500px 偏移量，当画布宽度不足 500px 或缩放后，箭头会出现在坐标轴中间而非端点处。箭头位置应基于实际画布尺寸动态计算。

3. 当前仅在坐标原点位于画布可见区域内时才绘制坐标轴线。用户平移画布使原点移出视野后，X 轴和 Y 轴会完全消失。坐标轴应始终穿过画布可见区域——X 轴始终是经过世界坐标 y=0 位置的屏幕水平线，Y 轴始终是经过世界坐标 x=0 位置的屏幕竖直线。

4. 右侧测量信息面板中，选中圆时只显示了半径长度，没有显示圆心坐标。提示词要求选中圆时同时显示半径长度和圆心坐标。

5. 绘制线段和圆的预览时，标注文字的字体大小写死为固定值，不随画布缩放比例变化。当画布缩放到 300% 或 50% 时，预览标注的文字大小与实际绘制图形的标注不一致。

6. 修复后需要确认单元测试仍然全部通过，如有失败需修正。测试代码放在 src/pages/__tests__/geometry-board/ 下。

7. 重新运行 pnpm run test 和 pnpm run lint，确保测试通过且无 ESLint 报错。修复完成后启动 pnpm run dev，对照提示词逐条验证每个功能的实际渲染效果。
