在当前项目中实现像素画编辑器，该页面是首页任务卡片"像素画编辑器"点击后的二级页面，所有绘制操作基于 Canvas 实现，使用 localStorage 保存当前编辑数据。

这个页面是一个像素风格的绘画工具，用户可以在像素网格中逐格填色来创作像素画。

1. 可配置像素网格：页面主体为 Canvas 渲染的像素网格区域，默认 16×16 的网格。用户可以修改网格尺寸——提供宽高输入框，范围 8×8 到 64×64，修改后网格重建。网格线使用浅灰色细线分隔每个像素格。Canvas 中的每个像素格大小随网格尺寸自适应计算。
2. 调色板：画布下方或右侧提供调色板，预设一组常用颜色（至少 20 种，包含黑白和基本色系）。当前选中的颜色有高亮边框。同时提供一个自定义颜色选择器（input type="color"），用户可以选择任意颜色并添加到调色板中。调色板支持添加和删除颜色。
3. 笔刷与填充工具：工具栏提供多种绘制工具——笔刷（点击单个像素格填色，按住拖拽连续绘制）、填充（点击某个像素格，将其所在连通区域全部替换为当前颜色）、橡皮擦（点击像素格将其恢复为空白/透明）。每种工具激活时图标高亮显示。提供笔刷大小调节（1×1、2×2、3×3 像素）。
4. 吸管工具：提供吸管工具按钮，激活后在画布上点击某个像素格，自动将该格颜色设置为当前绘制颜色，工具自动切换回之前使用的绘制工具。
5. 撤销与重做：顶部工具栏提供撤销和重做按钮，支持 Ctrl+Z / Ctrl+Y 快捷键。每次像素填色操作（包括笔刷一笔连续绘制、一次填充、一次橡皮擦擦除）记录为一个历史步骤。撤销可逐步回退，重做可逐步恢复。历史记录上限为 50 步。
6. 缩放预览：在画布旁边提供一个小型预览窗口，实时显示当前画布的缩小预览图（不受主画布缩放影响），帮助用户查看整体效果。主画布支持鼠标滚轮缩放（100%-1600%），缩放时像素格边界清晰可见（无模糊插值）。
7. 导出 PNG：点击「导出」按钮将当前画布内容导出为 PNG 图片下载。导出时考虑当前网格尺寸——如 16×16 的画布导出为 16×16 像素的 PNG（1:1 像素映射），也可以选择放大倍数导出（如 4x 放大导出 64×64）。导出前弹出选项对话框选择放大倍数。
8. JSON 数据导入导出：支持将当前画布数据（网格尺寸、每个像素格的颜色值）导出为 JSON 文件下载。支持从 JSON 文件导入恢复画布状态。导入时校验 JSON 格式是否正确，数据与当前网格尺寸是否匹配，不匹配时提示并取消导入。
9. 代码实现需要放在 src/pages/pixel-editor/ 下，测试代码放在 src/pages/__tests__/pixel-editor/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
