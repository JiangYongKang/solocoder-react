## 实现的功能

| 功能 | 描述 |
|------|------|
| **可配置像素网格** | 默认 16×16，支持 8×8 到 64×64，自适应像素格大小 |
| **调色板** | 预设 20 种颜色，支持自定义颜色选择器、添加/删除颜色 |
| **笔刷工具** | 点击/拖拽绘制，1×1、2×2、3×3 三种笔刷大小 |
| **填充工具** | 洪水填充连通区域 |
| **橡皮擦** | 恢复像素为透明/空白 |
| **吸管工具** | 点击像素拾取颜色，自动切回原工具 |
| **撤销/重做** | 支持 50 步历史记录，Ctrl+Z / Ctrl+Y 快捷键 |
| **缩放预览** | 实时缩略图预览 + 鼠标滚轮缩放（100%-1600%） |
| **导出 PNG** | 支持 1x、2x、4x、8x、16x 放大倍数导出 |
| **JSON 导入导出** | 完整画布数据（尺寸+颜色+调色板）导入导出，格式校验与尺寸不匹配提示 |
| **本地持久化** | 自动保存到 localStorage，刷新页面恢复状态 |
