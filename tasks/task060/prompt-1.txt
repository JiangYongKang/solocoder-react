在当前项目中实现在线简历编辑器，该页面是首页任务卡片"在线简历编辑器"点击后的二级页面，使用 localStorage 保存编辑中的简历数据。

这个页面是一个在线简历制作工具，用户可以选择模板、编辑简历内容并预览和导出。

1. 多模板选择：页面提供至少 3 种简历模板风格供用户选择（如经典、现代、简约）。模板以缩略图卡片形式展示，点击选中后简历内容和布局切换为对应模板渲染。模板切换时保留已填写的简历数据，仅改变布局和样式呈现。当前选中的模板卡片高亮标记。
2. 分区模块增删排序：简历按分区模块组织——基本信息、求职意向、教育经历、工作经历、项目经历、技能特长、自我评价等。左侧或顶部提供一个模块管理面板，列出所有模块，每个模块有显示/隐藏开关。模块支持拖拽排序，排列顺序决定在简历中的显示顺序。用户可以删除不需要的模块（基本信息模块不可删除）。支持添加自定义模块（输入模块名称后新增一个空白内容区）。
3. 表单编辑：点击任意简历模块进入编辑状态，模块内容以表单形式展示。「基本信息」模块包含姓名、联系电话、邮箱、所在地、个人主页等字段，联系电话和邮箱做格式校验。「教育经历」「工作经历」「项目经历」模块支持多条记录的增删——每条记录包含时间段、机构/公司名称、描述等字段，点击「添加一条」新增记录项。表单字段变化时右侧 A4 预览区实时同步更新。
4. A4 实时预览：页面右侧展示一张 A4 纸比例（210mm × 297mm）的预览区域，以白色卡片形式渲染当前的简历内容。预览区内简历按选中模板的布局和样式渲染，字体大小按比例缩放。预览支持滚动查看完整内容。编辑表单时预览内容实时同步。预览区域自适应页面高度，在 375px 移动端宽度下仍可完整查看。
5. PDF 模拟导出（可打印 HTML）：点击「导出」按钮，将当前简历内容渲染为一个新的浏览器窗口或隐藏 iframe 中的可打印页面。页面调用 window.print() 触发系统打印对话框，用户可选择「另存为 PDF」来保存为 PDF 文件。打印样式需专门处理——A4 纸张大小、页边距适配、隐藏编辑 UI 只显示简历内容、分页处理（如内容过长自动分页）。
6. JSON 导入导出：支持将简历数据（包括模板选择、模块配置、各模块填写内容）导出为 JSON 文件下载。支持从 JSON 文件导入恢复完整简历状态。导入时校验 JSON 格式和数据完整性，无效数据提示错误。
7. 代码实现需要放在 src/pages/resume-editor/ 下，测试代码放在 src/pages/__tests__/resume-editor/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
