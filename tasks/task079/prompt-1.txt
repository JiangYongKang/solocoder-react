在当前项目中实现 CSS 动画生成器，使用 localStorage 持久化动画数据。

这个页面是一个可视化的 CSS 关键帧动画编辑工具，用户可以通过时间轴编辑关键帧、配置动画属性、调整贝塞尔缓动曲线，实时预览动画效果，并导出 @keyframes 代码。

1. 关键帧时间轴可视化编辑：页面主体展示动画时间轴——横轴为时间百分比（0% 到 100%），刻度线标识 0%、25%、50%、75%、100%。时间轴上有多个轨道——每个 CSS 属性一条轨道（如 translateX、translateY、scale、rotate、opacity、background-color）。关键帧以圆点标记在对应轨道的对应时间位置，点击轨道空白处在该位置添加关键帧（自动插入该属性当前值的过渡）。关键帧点支持拖拽到不同时间位置。点击关键帧点选中后可在右侧属性面板编辑该关键帧的属性值。删除关键帧通过选中后按 Delete 键或右键菜单删除。同一属性至少保留 0% 和 100% 两个关键帧。
2. 多属性支持：支持编辑的动画属性包括——位移（translateX、translateY，单位 px 或 %）、缩放（scale，0.1 到 5）、旋转（rotate，单位 deg，0-360）、透明度（opacity，0-1）、背景颜色（background-color，支持颜色选择器）、边框圆角（border-radius，单位 px 或 %）、宽度/高度（width/height，单位 px 或 %）。属性列表支持添加和移除属性轨道。每个属性轨道左侧显示属性名标签，并有一个眼睛图标开关控制该属性是否在预览中生效。
3. 贝塞尔缓动编辑器：时间轴上每两个关键帧之间有一个缓动曲线段——默认 linear，点击线段可打开缓动编辑面板。面板中提供预设缓动函数——linear、ease、ease-in、ease-out、ease-in-out，以及一个 Cubic Bezier 可视化编辑器——一个正方形区域内显示贝塞尔曲线，两个控制点可拖拽调整，对应生成 cubic-bezier(p1x, p1y, p2x, p2y) 数值。编辑面板实时显示对应的 CSS transition-timing-function 代码。缓动设置应用于该段之间的过渡效果。
4. 实时预览：时间轴上方提供动画预览区——一个彩色的预览元素（方块或圆形），尺寸约 100x100px，应用当前编辑的动画效果。点击「播放」按钮运行动画——动画以设定的持续时间和缓动效果连续循环执行。提供「暂停」按钮和「重置」按钮。动画持续时长通过一个滑块设置——范围 0.5s 到 5s，默认 2s。提供播放速度调节——0.5x、1x、2x。迭代次数设置——infinite 或自定义次数。预览区背景显示网格辅助线，便于观察位移变化。
5. @keyframes 代码输出与复制：预览区下方提供代码输出面板——实时生成完整的 @keyframes 动画名称 { ... } CSS 代码，以及应用动画的 CSS 规则（animation 属性简写）。代码有语法高亮。面板上方提供动画名称输入框（默认 myAnimation）。提供「复制代码」按钮——一键复制全部 CSS 代码到剪贴板。复制后按钮显示「已复制」。
6. JSON 导入导出：工具栏提供「导出 JSON」按钮——将当前动画配置（所有关键帧数据、属性设置、缓动设置、动画参数）导出为 JSON 文件下载。「导入 JSON」按钮——选择之前导出的 JSON 文件恢复动画配置。导入时做基本的格式校验——如果 JSON 格式不合法或缺少必要字段，显示错误提示。
7. 动画列表管理：左侧边栏提供「我的动画」列表——展示所有保存的动画，每个显示动画名称、持续时长、属性数量、创建时间。支持点击加载动画到编辑器、重命名、删除。动画列表数据持久化到 localStorage。最多保存 20 个动画。
8. 代码实现需要放在 src/pages/css-animation/ 下，测试代码放在 src/pages/__tests__/css-animation/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
