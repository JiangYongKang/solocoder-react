在当前项目中实现成绩管理系统，使用 localStorage 持久化学生成绩数据。

这个页面是一个教学场景下的成绩管理工具，用户可以批量录入成绩数据、在学生视图和科目视图之间切换、查看统计分析结果和排名，并支持 CSV 导出。

1. 成绩表录入：页面主体展示成绩表格——行为学生姓名，列为科目（预设科目包括语文、数学、英语、物理、化学，支持添加/删除自定义科目）。支持两种录入方式——逐格编辑（点击单元格直接修改分数，失焦或回车确认，分数范围 0-150 分校验）和批量粘贴（表格上方提供文本输入区域，支持粘贴制表符分隔的多行成绩数据，粘贴后自动解析填充到表格中，解析时校验姓名列非空、分数列为数字）。表格首行固定，支持横向和纵向滚动。表格底部显示每个学生的总分和平均分汇总行。
2. 学生/科目双视图切换：提供学生视图（行为学生，列为科目，每行显示该学生各科成绩）和科目视图（行为科目，列为学生，每行显示该科目下所有学生成绩，适合横向比较单科成绩）。视图切换通过顶部 Tab 按钮实现。两种视图共享同一份数据源，切换视图后已输入数据不丢失。两种视图均支持逐格编辑。
3. 统计分析：提供统计分析面板——展示各科目的均分、最高分、最低分、中位数和标准差。统计数据以表格展示——每行一个科目，列显示统计指标。统计值保留两位小数。中位数计算需处理偶数个成绩时取中间两个值的平均。标准差使用总体标准差公式。
4. 直方图：统计分析面板下方展示成绩分布直方图——选择一个科目后，以柱状图展示该科目的分数段分布。分数段自动划分为 0-59（不及格）、60-69（及格）、70-79（中等）、80-89（良好）、90-100（优秀）、100+（超出满分标注为异常）。柱子高度表示该分数段的学生人数，柱子上方标注具体人数。直方图使用纯 CSS 或 Canvas 自绘。
5. 排名：排名展示区——按总分降序展示所有学生的排名、姓名、总分及各科成绩。同分学生的排名相同（跳过后续名次，如两个第 1 名后下一个是第 3 名）。排名表支持按单科成绩切换排名依据。排名变化用箭头标注（与上次保存数据对比）。
6. CSV 导出：工具栏提供「导出 CSV」按钮——将当前成绩表导出为 CSV 文件下载。CSV 第一行为表头（姓名、科目1、科目2、...、总分），后续每行为一个学生的完整数据。导出文件名为"成绩表_导出时间.csv"。
7. 代码实现需要放在 src/pages/grade-manager/ 下，测试代码放在 src/pages/__tests__/grade-manager/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
