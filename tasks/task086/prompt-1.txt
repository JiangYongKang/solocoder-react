在当前项目中实现截图标注工具，使用 Canvas 进行标注绘制，使用 localStorage 持久化标注数据和历史。

这个页面是一个图片标注工具，用户可以上传或粘贴截图到画布上，使用多种标注工具（箭头、矩形、圆形、文字、画笔）进行标注，支持元素的选中调整、撤销重做和标注数据的导出。

1. 上传或粘贴截图：页面顶部提供图片导入区域——「上传图片」按钮打开文件选择器（支持 png/jpg/jpeg/gif/webp 格式）、「粘贴图片」区域（用户从剪贴板 Ctrl+V 粘贴截图到此处，支持 Clipboard API 读取图片数据）。导入后图片绘制到 Canvas 画布上——图片保持原始宽高比适配画布（画布默认 900×600px），如果图片大于画布则等比缩放适配，小于画布则居中显示。提供「上传新图片」按钮替换当前图片。
2. 标注工具集：画布左侧提供标注工具栏——包含箭头工具（点击起点拖拽到终点，绘制带箭头头的线段）、矩形工具（点击起点拖拽到终点，绘制矩形框，支持圆角）、圆形/椭圆工具（点击起点拖拽到终点，绘制椭圆）、文字工具（点击画布位置弹出输入框，输入文字后渲染到该位置）、画笔工具（自由绘制路径，鼠标按下拖拽移动绘制线条）。工具栏中每种工具有图标+文字标签，当前选中工具高亮显示。标注元素的样式可配置——颜色选择器（预设红/黄/蓝/绿/白/黑 6 色+自定义颜色）、线条粗细选择（1px/2px/3px/5px，画笔工具专用）、文字字号选择（12px/16px/20px/24px）、矩形/圆形填充透明度（不填充/半透明/完全不透明）。
3. 元素选择与调整：画布上的标注元素支持点击选中——选中后元素显示调整手柄（矩形/圆形的四角拖拽点用于调整大小，箭头的两端拖拽点用于调整起止位置，文字的拖拽点用于移动位置）。选中元素高亮显示——矩形虚线框包围。选中状态下按 Delete 键或点击工具栏的「删除」按钮删除选中元素。支持多选——按住 Shift 点击多个元素，多选状态下可批量删除或移动。元素支持拖拽移动——点击选中后在元素区域内按住鼠标拖拽可移动位置。
4. 撤销与重做：工具栏提供撤销和重做按钮——撤销回退到上一步操作之前的状态（支持上传图片后的所有标注操作），重做恢复已撤销的操作。操作历史栈最多保留 50 步。按钮在无可撤销或可重做操作时显示为禁用状态。支持快捷键 Ctrl+Z 撤销和 Ctrl+Shift+Z 或 Ctrl+Y 重做。
5. 导出下载：工具栏提供「导出图片」按钮——将画布当前内容（原图+所有标注）导出为 PNG 文件下载，文件名格式为"标注截图_导出时间.png"。导出分辨率为画布当前实际像素尺寸。导出时标注元素以矢量方式合并渲染。
6. 标注数据 JSON 保存恢复：工具栏提供「保存标注」和「加载标注」功能——保存时将当前所有标注元素的数据（类型、坐标、颜色、粗细、文字内容等）序列化为 JSON 保存到 localStorage，同时保存关联的图片信息（图片数据以 Base64 存储，如果图片过大超过 2MB 则从 localStorage 中单独存储索引）。加载时从已保存的标注列表中选择恢复——标注列表显示保存时间、标注元素数量。同一张图片可以有多个标注版本。支持重命名和删除已保存的标注数据。
7. 代码实现需要放在 src/pages/screenshot-annotator/ 下，测试代码放在 src/pages/__tests__/screenshot-annotator/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑。
