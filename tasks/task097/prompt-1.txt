在当前项目中实现导航菜单设计器页面，使用 localStorage 持久化所有菜单配置。

这个页面是一个可视化导航菜单编辑工具，用户可以拖拽编辑多级菜单结构、编辑每个菜单项的属性、切换布局方案并实时预览效果，以及导出菜单配置为 JSON。

1. 多级菜单树形拖拽编辑：页面左侧为菜单树形编辑器——以树形列表展示菜单的层级结构（支持无限层级嵌套）。每个节点显示菜单名称和类型标识（链接/菜单组/分割线）。支持拖拽节点调整顺序——在同一层级内上下拖拽排序，或拖动到其他节点下方作为子节点（缩进显示）。支持右键菜单操作——添加子菜单、在同级上方/下方添加菜单、编辑、删除。删除父菜单时需确认——其下所有子菜单一并删除。
2. 属性编辑：点击树形中的菜单节点后，右侧显示属性编辑面板——编辑菜单名称（文本输入框）、图标（下拉选择预设图标列表，如 home/user/settings/mail/bell/folder/file/search/star/heart/gear，图标使用 Unicode 符号或 emoji 展示）、链接地址（URL 输入框，以 # 开头的锚点或完整 URL）、权限要求（文本输入，如 "admin"、"user"，为空表示无限制）、打开方式（下拉——当前页/新标签）。属性修改后左侧树形节点和右侧预览实时更新。
3. 横向/纵向/折叠三种布局：顶部提供布局切换——横向（顶部水平导航栏样式）、纵向（左侧垂直菜单栏样式）、折叠（左侧可折叠的垂直菜单，父菜单点击展开/收起子菜单的折叠面板样式）。右侧预览区实时切换布局展示。横向布局下二级菜单以下拉菜单形式展示。每种布局使用不同的 CSS 样式——横向为顶部横条 + 下拉，纵向为左侧全高侧边栏，折叠为左侧手风琴面板。
4. 实时预览：页面右侧为菜单实时预览面板——展示当前菜单配置在选中布局下的渲染效果。预览面板宽度模拟真实场景——横向布局约 800px 宽，纵向和折叠布局约 250px 宽。预览中的菜单项可点击展开子菜单（在横向布局中为悬停下拉，纵向和折叠中为点击展开）。预览区域标题栏显示当前导航标题（如"后台管理系统"）。
5. JSON 导出：提供"导出 JSON"按钮——将当前菜单配置导出为 JSON 文件下载。JSON 结构包含菜单树形数据（name/icon/link/target/permission/children 字段）和布局类型。提供"导入 JSON"按钮——从本地 JSON 文件加载菜单配置覆盖当前编辑器内容。导入前校验 JSON 格式合法性。
6. 代码实现需要放在 src/pages/menu-designer/ 下，测试代码放在 src/pages/__tests__/menu-designer/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如菜单树形操作、JSON 导入导出校验、拖拽排序逻辑等）。
