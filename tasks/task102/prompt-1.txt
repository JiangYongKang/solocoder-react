在当前项目中实现 Markdown 简历模板市场页面，所有模板数据和用户配置在前端维护，支持 localStorage 持久化。

这个页面是一个简历模板浏览与编辑平台，用户可以从多个风格的简历模板中选择并应用，在线编辑简历内容并实时预览 Markdown 渲染效果。

1. 多风格模板预览：页面顶部展示模板选择区——以卡片横向滚动或网格方式展示至少 6 种不同风格的简历模板（如经典黑白、现代蓝白、极简线条、创意卡片、双栏布局、深色主题等）。每张卡片显示模板的缩略预览图（可用纯色占位图或 CSS 绘制的微缩布局示意）和模板名称。当前选中的模板卡片以边框高亮标记。
2. 一键应用模板：点击模板卡片后，右侧 Markdown 预览区立即切换为该模板的样式渲染。模板切换时保留用户已编辑的简历内容——只改变展示风格和排版布局，不清空已填内容。切换模板有一个短暂的过渡动画。
3. 模块拖拽排序：页面分为左右两栏——左侧为模块编辑器，右侧为实时预览。左侧编辑区以可拖拽的模块卡片列表展示简历的各个部分（如个人信息、教育经历、工作经历、项目经验、技能标签、自我评价等）。用户可以拖拽模块卡片上下调整模块在简历中的显示顺序。每个模块卡片有开关按钮——关闭后该模块在预览中隐藏但不删除内容。模块卡片可以展开收起——展开后编辑该模块的具体内容。
4. 实时 Markdown 渲染：每个模块内部的内容以 Markdown 格式编辑（使用 textarea 或富文本编辑区），右侧预览区实时将 Markdown 渲染为 HTML 并应用当前选中模板的 CSS 样式。Markdown 支持常规语法——标题、列表、粗体斜体、链接、代码块、分割线。预览区渲染效果接近实际打印效果——A4 纸比例展示。
5. PDF 模拟打印：提供"打印/导出 PDF"按钮——点击后打开浏览器打印对话框，打印时只显示简历预览内容（隐藏页面其他 UI 元素），采用 A4 尺寸的打印样式。页面内也可以切换到"打印预览"模式——全屏展示 A4 纸的简历预览，模拟最终打印效果。
6. 模板收藏与评分：每个模板卡片上显示星级评分（1-5 星）和收藏按钮。用户可以给模板打分和切换收藏状态。页面提供"收藏/全部"筛选切换，筛选显示已收藏的模板。评分和收藏状态持久化到 localStorage。
7. 代码实现需要放在 src/pages/resume-templates/ 下，测试代码放在 src/pages/__tests__/resume-templates/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如 Markdown 解析转换、模块排序更新、模板数据结构、评分过滤、收藏状态管理等）。
