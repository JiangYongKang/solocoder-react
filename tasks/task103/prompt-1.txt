在当前项目中实现网络拓扑图编辑器页面，所有拓扑数据在前端维护，支持 JSON 保存与恢复和 PNG 导出。

这个页面是一个可视化的网络拓扑图编辑工具，用户可以通过拖拽添加网络设备节点、绘制连线来构建网络架构图，支持自动布局和画布操作。

1. 设备节点拖拽：页面左侧提供设备节点面板——以图标或卡片形式列出至少 6 种设备类型（如服务器、路由器、交换机、防火墙、工作站、云服务）。每种设备类型有独特的外观标识（如不同的形状或颜色——服务器用矩形蓝色、路由器用圆形橙色、交换机用方形绿色等）。用户从面板中拖拽设备到右侧画布上放置，画布上可自由拖拽移动已放置的设备节点调整位置。每个节点显示设备名称标签。
2. 连线与端口配置：点击选中画布上的一个设备节点后，显示连接端口（如设备边缘出现的连接点小圆点）。从一个设备的端口拖拽到另一个设备的端口即可创建连线。连线为直线或贝塞尔曲线。每条连线选中后可设置线型（实线/虚线）、线宽、标签文字（如"千兆光纤"、"VPN 隧道"）。选中连线后按 Delete 键删除。
3. 自动布局算法：工具栏提供"自动布局"按钮——点击后将画布上所有节点按照层次布局或力导向布局算法自动重新排列。布局后节点之间保持合理的间距避免重叠，连线自动跟随节点位置更新。提供"垂直布局"和"水平布局"两种自动布局方向选择。
4. 缩放平移：画布支持鼠标滚轮缩放（放大/缩小）和鼠标拖拽平移画布。缩放时节点和连线同步缩放。提供画布缩放到适合窗口的"适应画布"按钮和"重置缩放"按钮。画布右下角显示当前缩放百分比。
5. JSON 保存：工具栏提供"保存"和"加载"按钮——保存时将当前拓扑结构（包括所有节点位置、连线、属性配置）序列化为 JSON 字符串下载为文件。加载时上传之前保存的 JSON 文件恢复拓扑图。页面刷新后拓扑图从 localStorage 自动恢复。
6. 导出 PNG：工具栏提供"导出图片"按钮——将当前画布区域的拓扑图导出为 PNG 图片下载。导出的图片只包含拓扑图内容，不包含工具栏和设备面板等 UI 元素，以白色背景导出。
7. 代码实现需要放在 src/pages/network-topology/ 下，测试代码放在 src/pages/__tests__/network-topology/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如自动布局算法、拓扑数据序列化与反序列化、连线创建与删除逻辑、节点位置更新等）。
