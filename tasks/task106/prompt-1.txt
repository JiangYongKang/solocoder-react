在当前项目中实现 SVG 路径编辑器页面，编辑器全部基于 Canvas 或 SVG 原生元素实现。

这个页面是一个可视化的 SVG 路径编辑工具，用户可以通过拖拽贝塞尔曲线控制点来编辑路径形状，管理多个路径，并导出标准 SVG 代码。

1. 贝塞尔曲线控制点拖拽：页面中央为 SVG 或 Canvas 编辑画布，画布上显示当前正在编辑的路径——路径以贝塞尔曲线渲染。路径上每个节点以可拖拽的圆形控制点显示——锚点以实心圆表示，控制柄端点以空心圆表示。用户可以拖拽锚点移动节点的位置，拖拽控制柄端点调整曲线的弯曲方向和弧度。鼠标悬停在控制点上时光标变为抓取样式。拖拽锚点时两条相邻的贝塞尔曲线段同步更新。
2. 路径命令面板：编辑画布右侧或底部显示路径命令面板——以表格或列表形式展示当前路径的所有 SVG 路径命令（M 移动到、L 直线到、C 三次贝塞尔曲线、Q 二次贝塞尔曲线、A 弧线、Z 闭合路径）。每条命令显示命令字母和参数值，如 `M 100 200`、`C 150 180 200 250 250 200`。用户可以直接在面板中编辑命令参数数值，画布上的路径同步更新。面板也提供"A 弧线"命令的辅助参数输入（rx、ry、x-axis-rotation、large-arc-flag、sweep-flag），让用户无需记忆弧线参数格式。
3. 节点增删：工具栏提供"添加节点"模式——在路径某线段上点击即可在该位置插入一个新节点，原有的单段贝塞尔曲线自动拆分为两段并保持形状连续。选中一个已有节点后按 Delete 键或点击"删除节点"按钮删除该节点，相邻两个节点之间自动生成新的连接曲线。提供"转为直线"按钮——将选中节点与下一个节点之间的曲线段转为直线段（C/Q 命令变为 L 命令）。
4. 填充与描边样式：编辑画布侧边提供样式面板——设置路径的填充颜色（纯色/渐变/无填充）、描边颜色、描边线宽、描边端点样式（平头/圆头/方头）、描边连接样式（尖角/圆角/斜角）、虚线样式（实线/自定义虚线数组）。修改样式后画布上的路径实时更新。样式面板中提供颜色选择器（预设颜色板 + 自定义 HEX 输入）。
5. 路径列表管理：页面左侧显示路径列表侧边栏——列出当前编辑的所有路径条目，每条显示路径名称（可双击重命名）和缩略预览（微缩的路径形状）。用户可以新增路径、复制路径、删除路径、上下拖拽调整路径顺序。选中某条路径后画布切换到该路径的编辑视图。提供"显示/隐藏"眼球图标切换路径在画布上的可见性。
6. 导出 SVG 代码：工具栏提供"导出 SVG"按钮——点击后将当前选中的路径（或全部路径）生成为完整的 SVG 代码字符串。导出的 SVG 代码显示在弹窗中，包含 `<svg>` 标签包裹的 `<path d="...">` 元素和样式属性，代码带语法高亮。提供"复制代码"按钮一键复制到剪贴板。提供"下载 SVG 文件"按钮下载为 .svg 文件。
7. 代码实现需要放在 src/pages/svg-path-editor/ 下，测试代码放在 src/pages/__tests__/svg-path-editor/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如 SVG 路径命令解析与序列化、节点插入与删除后的坐标重算、贝塞尔曲线参数转换、样式对象合并等）。
