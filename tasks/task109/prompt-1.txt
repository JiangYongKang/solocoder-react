在当前项目中实现分享卡生成器页面，所有图片基于 Canvas 渲染生成。

这个页面是一个社交媒体分享卡片设计工具，用户可以自定义背景、文字、Logo 和二维码来制作精美的分享卡片，支持多模板和 PNG 导出。

1. 背景设置：页面左侧为编辑面板，右侧为 Canvas 实时预览区（预览区以移动端分享卡比例展示，如 1:1 方形或 16:9 横版）。编辑面板中"背景"区提供三种背景模式切换——背景图（本地上传图片，铺满整个卡片，可调节模糊度和叠加颜色）、渐变色（支持双色渐变，选择起始色和结束色，渐变方向可选择水平/垂直/对角线）、纯色（从预设色板选择或输入 HEX 色值）。三种模式可随时切换，切换时保留其他设置不变。
2. 内容编辑：编辑面板中"内容"区提供文本编辑——标题输入框（单行，字体较大，位于卡片上部区域）、描述输入框（多行文本，字体较小，位于标题下方）。提供字体样式调节——字体大小滑块、字体颜色选择器、字体对齐方式（左对齐/居中/右对齐）、粗体开关。所有调节在右侧 Canvas 预览区实时反映。
3. 二维码与 Logo 叠加：编辑面板中"元素"区提供二维码和 Logo 的添加——"上传 Logo"按钮（从本地选择图片，显示在卡片顶部或底部的固定位置区域，可调节 Logo 大小，Logo 自动保持宽高比）。"生成二维码"——在输入框中输入 URL 或文本，点击生成后在 Canvas 上以二维码图片形式渲染在卡片底部区域。二维码使用 Canvas 原生绘制（不使用外部二维码库，通过 Canvas 像素绘制 QR 码的简化模拟——绘制黑白方块矩阵表示一个二维码样式图案）。Logo 和二维码可分别开关显示。
4. 多模板选择：编辑面板顶部提供模板选择——多个预设模板缩略图横向排列，每个模板包含预设的背景色/渐变、标题和描述的字体大小和颜色、Logo 和二维码的默认位置。点击模板后所有设置切换为该模板的默认配置，但用户可以在此基础上自由修改。当前选中的模板以边框高亮。
5. Canvas 渲染导出 PNG：右侧 Canvas 预览区实时绘制分享卡的最终效果——Canvas 尺寸为 600×600 像素（方形卡片）或 600×1000 像素（竖版卡片）。Canvas 上分层绘制——最底层的背景（图片/渐变色/纯色）、中间层的文字（标题和描述）、上层的 Logo 和二维码。提供"导出 PNG"按钮——将当前 Canvas 内容导出为 PNG 图片触发浏览器下载，文件名默认为"share-card-时间戳.png"。提供"复制图片"按钮——将 Canvas 内容复制到剪贴板。
6. 配置 JSON 保存：提供"保存配置"按钮——将当前所有编辑设置（背景模式与参数、文本内容与样式、Logo 位置与大小、二维码内容、模板选择）序列化为 JSON 保存到 localStorage。提供"加载配置"按钮——从 localStorage 中恢复之前保存的配置。配置列表以保存时间命名展示，可删除旧配置。页面刷新后自动恢复上次编辑的配置。
7. 代码实现需要放在 src/pages/share-card/ 下，测试代码放在 src/pages/__tests__/share-card/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如配置 JSON 序列化与反序列化、模板默认配置数据结构、Canvas 绘制坐标计算、背景模式切换的状态管理等）。
