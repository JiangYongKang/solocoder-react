在当前项目中实现弹窗生成器页面，提供一个可视化的弹窗配置和预览工具。

这个页面让用户通过可视化面板配置弹窗的各个属性和样式，实时预览弹窗效果，并生成对应的调用代码以供复制使用。

1. 弹窗类型选择：页面顶部提供弹窗类型选择——确认弹窗（Confirm，含确定/取消按钮）、提示弹窗（Alert，仅确定按钮）、表单弹窗（Form，包含输入框和提交/取消按钮）、信息弹窗（Info，纯展示无按钮或仅关闭按钮）。切换类型后，右侧预览区即时显示对应类型的弹窗结构。表单弹窗支持配置输入框数量（1-5 个），每个输入框可设置标签文字和占位符。
2. 标题内容按钮自定义：编辑面板的"内容"区提供——弹窗标题输入框（如"确认删除"）、弹窗正文内容输入框（支持多行文本）、确定按钮文字（默认为"确定"）、取消按钮文字（默认为"取消"）。按钮颜色可自定义——确定按钮颜色（预设色板 + HEX 输入）、取消按钮颜色。弹窗宽度支持通过滑块从 300px 到 800px 调节。
3. 遮罩透明度与点击关闭开关：编辑面板的"遮罩"区提供——遮罩透明度滑块（0% 到 100%，预览区遮罩层透明度实时变化）、"点击遮罩关闭"开关（控制点击弹窗外部遮罩区域是否关闭弹窗）。提供"显示关闭按钮"开关——控制弹窗右上角是否显示 X 关闭按钮。
4. 入场动画选择：编辑面板的"动画"区提供入场动画选择——从预设动画列表中选取（如淡入、从上方滑入、从下方滑入、缩放进入、从左侧滑入、从右侧滑入）。选择动画后点击"预览动画"按钮观看弹窗入场效果。动画时长通过滑块从 100ms 到 1000ms 调节。
5. 调用代码复制：编辑面板底部提供"生成代码"按钮——根据当前所有配置动态生成调用该弹窗的 JavaScript 代码。代码展示在代码面板中，附带语法高亮。代码示例形如 `openModal({ type: 'confirm', title: '...', content: '...', ... })`。"复制代码"按钮将生成的代码复制到剪贴板，复制成功后按钮短暂变为"已复制"状态。
6. 预览区域：页面右侧为弹窗实时预览区——预览区模拟一个页面背景，弹窗居中显示。预览区中弹窗的所有属性实时反映编辑面板的修改。提供"全屏预览"按钮——在浏览器窗口中以全屏方式预览弹窗效果，按 Esc 键退出全屏。
7. 代码实现需要放在 src/pages/modal-generator/ 下，测试代码放在 src/pages/__tests__/modal-generator/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如弹窗配置对象生成、调用代码字符串构建、动画配置映射、类型切换时的默认属性重置等）。
