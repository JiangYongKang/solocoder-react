你实现的拼图游戏页面还存在一些问题，需要进行修复：

1. 拼块交换的步数计数逻辑不符合要求。当前每次交换都无条件增加步数，包括已经拼对的拼块被挪走再挪回来的无效操作。puzzleCore.js 中已经实现了 countIncorrectAfterSwap 函数用于判断"两块都不在正确位置才计一次交换"，但 swapPieces 函数从未调用它。需要在组件层集成该判断逻辑。

3. 暂停遮罩层及其内部文案使用内联 style 对象编写样式，而非 CSS 类名，导致样式不便集中管理。需要将暂停遮罩和暂停文案的样式提取到 CSS 文件中。
4. 运行 pnpm run test，需要让所有测试通过。
