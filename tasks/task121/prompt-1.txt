在当前项目中实现「颜色工具集」页面。该页面是首页任务卡片「颜色工具集」点击后的二级页面，使用 localStorage 做收藏数据持久化。

这个页面是一个面向前端开发者的颜色工具箱，用户可以生成调色板、进行 HEX/RGB/HSL 格式互转、色盲模拟预览、生成渐变色 CSS 代码、提取品牌色，并支持收藏和导出。

1. 调色板生成：用户选择一个基准色（支持 HEX 输入和颜色面板），页面自动生成三类调色板——互补色调色板（基准色及其色轮对面 180° 的颜色和变体）、类似色调色板（基准色及其色轮相邻 ±30° 的两个颜色和变体）、三色调色板（基准色及其色轮 ±120° 的两个颜色和变体）。每类调色板以一行色块展示，每个色块显示颜色预览和 HEX 值。点击色块将 HEX 复制到剪贴板，并短暂显示「已复制」提示。三类调色板通过 tab 切换或同页面展开查看。
2. HEX/RGB/HSL 互转：用户在一个输入框中输入 HEX（如 #FF5733）、RGB（如 rgb(255, 87, 51)）或 HSL（如 hsl(9, 100%, 60%））格式的颜色值，页面自动识别格式并实时转换为另外两种格式展示在下方。输入格式不合法时给出「格式无效」提示——分别说明 HEX、RGB、HSL 的合法格式要求。转换结果旁提供复制按钮，同时展示一个颜色预览块显示当前颜色。
3. 色盲模拟预览：用户选择一个颜色后，页面在 Canvas 上渲染原色块和三种色盲模拟色块——红色盲（Protanopia）、绿色盲（Deuteranopia）、蓝黄色盲（Tritanopia）。原色在上方，下方依次并排展示三种色盲视角下的颜色效果。每个模拟色块下方标注类型名称和对应的 HEX 值。色盲模拟算法通过 RGB 矩阵变换实现不同色盲类型的通道混合计算。
4. 渐变色生成 CSS 代码：提供起始色和结束色两个颜色选择器，渐变类型支持 linear-gradient 和 radial-gradient。linear-gradient 支持八个方向选择（上/下/左/右/对角线）。预览区实时展示渐变效果，下方实时输出对应的 CSS 代码（如 `background: linear-gradient(to right, #FF5733, #33FF57);`）。提供「复制 CSS」按钮。
5. 品牌色提取：用户上传一张品牌 Logo 或图片（png、jpg、webp），页面使用 Canvas 分析像素颜色分布，提取出现频率最高的 5 种主要颜色作为品牌色。结果以五个色块从主色到辅色依次排列，显示 HEX 值和占比百分比。点击可将色值加入收藏。上传非图片文件时提示错误。
6. 收藏导出：所有工具生成的色块旁提供「收藏」按钮（五角星图标），点击加入收藏列表。收藏列表展示色块、HEX 值、来源工具标签、收藏时间。支持拖拽排序、删除单个、一键清空。收藏数据持久化到 localStorage。提供「导出收藏」——将收藏导出为 JSON 文件下载。提供「复制全部 HEX」——将所有收藏 HEX 值以逗号分隔复制。
7. 代码实现需要放在 src/pages/color-toolkit/ 下，测试代码放在 src/pages/__tests__/color-toolkit/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如 HEX/RGB/HSL 互转算法、色盲模拟的颜色矩阵变换、调色板生成的色轮角度计算、品牌色提取的像素频率统计算法等）。
