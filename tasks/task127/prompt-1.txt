在当前项目中实现「加载动画生成器」页面。该页面是首页任务卡片「加载动画生成器」点击后的二级页面，使用 localStorage 做动画配置持久化。

这个页面是一个加载动画的可视化配置和代码生成工具，用户可以选择多种动画类型，调节颜色、大小和速度，实时预览动画效果，组合多个动画元素，并导出对应的 CSS 代码。

1. 旋转/脉冲/波浪/骨架屏等多类型动画：页面左侧为动画类型选择区——以卡片网格展示多种类型供选择——旋转加载（Spinner，圆形边框旋转）、脉冲加载（Pulse，圆点缩放呼吸）、波浪加载（Wave，多个竖条依次伸缩波动）、骨架屏（Skeleton，矩形块渐变模拟内容加载）、三点跳动（Dots，三个圆点依次弹跳）、进度条（Progress Bar，条状从左到右填充）、圆环进度（Circle Progress，圆形环填充）。每种显示缩略图预览。点击某类型后进入编辑和预览模式。
2. 颜色大小速度调节：选中类型后右侧编辑面板提供参数——颜色（主色和辅色的颜色选择器）、大小（动画元素尺寸，范围 10px-200px 滑块）、速度（动画持续时间，范围 0.3s-5s 滑块，默认 1s）、粗细（如旋转加载边框粗细、波浪条宽度，范围 1px-10px）、数量（如波浪条个数、圆点个数，范围 3-10）。调节时左侧预览区实时更新。
3. CSS 代码实时输出复制：编辑面板下方代码输出区——实时生成当前动画的完整 CSS 代码（含 HTML 注释、@keyframes 定义和 CSS 类）。代码以语法高亮展示。提供「复制代码」——一键复制全部 CSS，按钮显示「已复制」。提供「复制 HTML」——单独复制 HTML 结构片段。生成的 CSS 使用语义化类名（如 .spinner、.pulse-dot、.wave-bar）。
4. 预览：页面中央预览区展示当前动画的实际运行效果。预览区背景可切换——浅色和深色两个选项模拟不同场景效果。提供「播放/暂停」控制动画运行。预览区尺寸可调节——小（200×200）、中（400×400）、大（600×600）。背景显示网格辅助线。
5. 组合动画编辑器：提供组合动画功能——在组合编辑器画布上，用户从左侧动画类型中拖拽动画元素到画布，每个元素可独立调节位置（x、y 坐标）、颜色、大小、速度。画布上可添加多个元素（如旋转加载在左上角+波浪条在下方）。组合动画也支持预览和 CSS 代码导出——导出时生成包含所有子动画的完整 HTML 和 CSS。
6. JSON 保存：提供「保存」——将当前动画配置和组合动画数据序列化为 JSON 保存到 localStorage。提供「我的动画」列表——展示所有保存的配置，显示动画类型名称、缩略图预览、保存时间。列表支持加载到编辑器和删除。最多保存 20 个配置。
7. 代码实现需要放在 src/pages/loading-animation/ 下，测试代码放在 src/pages/__tests__/loading-animation/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如动画配置 JSON 序列化反序列化、CSS 代码生成函数、不同类型动画的 @keyframes 字符串构建函数等）。
