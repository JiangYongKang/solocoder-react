在当前项目中实现「应用启动页配置器」页面。该页面是首页任务卡片「应用启动页配置器」点击后的二级页面，使用 localStorage 做配置持久化。

这个页面是一个应用启动页（Splash Screen）的可视化配置工具，用户可以编辑 Logo、标题和副标题，选择背景和交互元素，套用模板快速配置，并通过全屏预览查看最终效果，导出配置 JSON。

1. Logo、标题、副标题编辑：页面左侧编辑面板——「品牌」区域包含 Logo 上传（选择本地图片，支持 png、jpg、svg，大小限制 1MB 超出提示）、标题输入框（单行，应用名称）和副标题输入框（单行，Slogan）。Logo 可调节显示尺寸（30px-150px 滑块）。标题和副标题可独立调节字体大小（12px-60px）、字体颜色、粗体开关。
2. 背景色/图片选择：「背景」区域提供三种背景模式——纯色（预设色板选择或输入 HEX）、背景图片（上传本地图片，支持填充/适应/拉伸三种模式）、渐变背景（双色渐变，选择起始色和结束色，渐变方向可选水平/垂直/对角线）。三种模式可切换，切换时保留已设置的色值或图片。
3. 倒计时/跳过按钮：「交互」区域提供倒计时设置——开启/关闭开关、倒计时秒数（1-10 秒，默认 3 秒）、文字格式（输入模板，用 {n} 占位秒数，如"跳过 {n}s"）。提供跳过按钮——开启/关闭开关、按钮文字、位置选择（右上角/右下角/底部居中）、颜色自定义。在预览区实时显示交互元素效果。
4. 多模板预览：编辑面板顶部提供模板选择——预设多种风格启动页模板（如极简白底居中 Logo、科技感深色背景、商务风顶部 Logo 底部文字、活泼风多彩渐变背景）。模板以缩略图网格展示。点击模板后所有配置切换为该模板默认值，用户可在此基础上修改。当前选中模板高亮。切换模板时提示「切换模板将覆盖当前配置，是否继续？」。
5. 配置 JSON 导出：「保存与导出」区域提供「保存配置」——将当前配置序列化为 JSON 保存到 localStorage，保存时需命名。配置列表展示所有已保存配置，支持加载、删除、重命名。提供「导出 JSON」——下载为 JSON 文件。提供「导入 JSON」——选择 JSON 文件恢复配置，校验格式和必要字段。
6. 全屏预览：提供「全屏预览」——进入全屏模式，按手机屏幕比例（375×812 iPhone 比例或 360×640 Android 比例可选）展示完整启动页效果。Logo、标题、副标题、背景、倒计时和跳过按钮按配置渲染。倒计时实际运行——从设置秒数递减到 0，或点击跳过按钮退出。Logo 进入时带淡入动画。按 ESC 键或「退出预览」按钮退出全屏。
7. 代码实现需要放在 src/pages/splash-config/ 下，测试代码放在 src/pages/__tests__/splash-config/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如配置 JSON 序列化反序列化、倒计时秒数校验、Logo 大小范围校验、背景模式切换的状态管理、模板默认配置数据结构等）。
