你实现的用户反馈收集系统还存在一些问题，需要进行修复：

1. 趋势图在部分浏览器中无法显示：TrendChart 组件使用了 Canvas 2D 的 roundRect API 绘制圆角矩形柱状图，该 API 在 Chrome 99、Firefox 112、Safari 15.4 以下的浏览器中不可用，导致整个趋势图表区域在这些浏览器中静默空白。

2. 搜索功能范围超出需求：searchByKeyword 函数同时搜索了标题和描述两个字段，而提示词要求的是"按关键词搜索标题"。搜索框的 placeholder 也写成了"搜索标题或描述..."，与需求不一致。

3. 统计总览数字的布局位置与提示词要求不符：提示词要求统计图右侧展示总览数字，但代码将四个统计卡片放在了趋势图的上方，用户在对比趋势图和总览数据时需要来回滚动。

4. 修复完成后重新运行 pnpm run test，需要让所有测试通过。
