在当前项目中实现「定位器网格编辑器」页面，使用 localStorage 做数据持久化。

这个页面是一个可视化 CSS 网格布局编辑工具，用户可以配置网格行列数，拖拽合并拆分单元格，设置对齐方式和边框样式，并一键生成对应的 HTML/CSS 代码。

1. 页面主体为画布区——格子网格可视化展示，初始默认 3 列 × 3 行。顶部配置栏提供「列数」和「行数」输入框——支持 1-12 之间的数值，修改后网格实时更新。提供「行高」和「列宽」像素值调节滑块（范围 40px-200px），所有行列统一尺寸。
2. 单元格拖拽合并拆分：支持鼠标拖拽选中多个相邻单元格——拖拽选区形成矩形高亮，释放后弹出菜单选择「合并单元格」（被合并区域合并为一个跨行列单元格）或「取消」。已合并的单元格右键点击弹出菜单——「拆分单元格」将其还原为原始行列数的最小单元格。合并后的单元格显示合并范围标记（如 "2×1"）。
3. 每个单元格支持设置对齐方式——选中单元格后，右侧属性面板提供水平对齐（左/中/右）和垂直对齐（上/中/下）选择。同时支持设置边框样式——边框颜色、边框粗细（1-5px）、边框样式（实线/虚线/点线）。设置实时应用到选中单元格。支持按住 Ctrl 或 Cmd 键多选单元格批量设置。
4. 顶部工具栏提供「显示网格线」和「隐藏网格线」切换——显示网格线时画布显示灰色网格线，隐藏时仅显示单元格内容边框。提供单元格「自动编号」功能——开启后每个单元格内显示其位置编号（如 "(1,2)" 表示第 1 列第 2 行）。
5. 代码生成导出：右侧面板底部展示实时生成的 HTML/CSS 代码预览——使用 CSS Grid 布局实现。HTML 生成每个单元格的 div 结构，CSS 包含 grid-template-columns、grid-template-rows、每个跨行列单元格的 grid-column/grid-row、对齐和边框样式。提供「复制代码」按钮一键复制。
6. 代码实现需要放在 src/pages/grid-editor/ 下，测试代码放在 src/pages/__tests__/grid-editor/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如网格合并拆分的数据结构变更、行列数变更后溢出单元格的处理、CSS Grid 代码生成逻辑、对齐和边框样式的配置映射等）。
