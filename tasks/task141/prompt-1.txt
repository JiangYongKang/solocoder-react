在当前项目中实现「网格布局生成器」页面，使用 localStorage 做配置持久化。

这个页面是一个 CSS Grid 可视化布局生成工具，用户可以配置网格的行数和列数，调节间距与对齐方式，为每个单元格独立设置跨行跨列属性，实时输出 CSS Grid 代码，并提供可视网格着色辅助和 JSON 保存功能。

1. 行数列数配置：页面顶部为网格配置面板——包含行数输入框（范围 1-12）和列数输入框（范围 1-12），通过加减按钮或直接输入调整。行高和列宽可分别配置——提供「固定像素」「百分比」「fr 单位」三种模式切换，每行每列可逐个设置不同数值或统一设置。配置变更后下方网格画布实时更新。
2. 间距与对齐调节：配置面板中包含行间距（row-gap）和列间距（column-gap）滑块，范围 0-50px。对齐方式部分提供三个下拉选择器——主轴对齐（justify-content：start/center/end/stretch/space-between/space-around/space-evenly）、交叉轴对齐（align-content：start/center/end/stretch/space-between/space-around/space-evenly）、单元格内部对齐（place-items：start/center/end/stretch）。每个选项旁有微小示意图标帮助理解。
3. 各单元格独立跨行列设置：网格画布中的每个单元格支持点击选中——选中后右侧属性面板展示该单元格的跨行列设置，包含「跨列数」和「跨行数」下拉框（范围 1-当前剩余列/行数）。设置跨列跨行后网格中该单元格视觉上合并，其他单元格自动重排。属性面板同时显示该单元格的网格区域名称（grid-area 自定义命名）和网格线编号（起始行线/结束行线/起始列线/结束列线）。
4. CSS Grid 代码输出：页面底部或右侧展示 CSS 代码面板——实时生成完整的 CSS Grid 样式代码，包含 grid-template-columns、grid-template-rows、gap、justify-content、align-content 和各单元格的 grid-area 定义。代码面板使用等宽字体语法高亮。提供「复制 CSS」按钮一键复制到剪贴板，以及「按需只复制容器样式」或「复制全部含子项样式」两种复制选项。
5. 可视网格着色辅助：网格画布中的每个单元格支持右键设置背景色——弹出颜色选择器选色。提供预设调色板（6-8 种柔和的网格配色）和自定义颜色。着色后的单元格在 CSS 中对应生成 background-color 样式。提供「清除所有颜色」和「随机着色」按钮。网格线编号（行线 1-13、列线 1-13）在画布边缘显示。
6. JSON 保存：顶部工具栏提供「保存布局」按钮——将当前所有配置（行列数、行列宽高、间距、对齐方式、各单元格的跨行列设置、颜色、区域名称）序列化为 JSON 并保存到 localStorage。提供「加载布局」按钮——从 localStorage 读取已保存的布局列表，选择一个恢复。支持「导出 JSON 文件」下载和「导入 JSON 文件」上传。布局名称可自定义，保存时记录时间戳。
7. 代码实现需要放在 src/pages/grid-generator/ 下，测试代码放在 src/pages/__tests__/grid-generator/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如 CSS Grid 代码生成器、配置序列化与反序列化、单元格跨度校验算法、行列尺寸格式化函数、JSON 导入校验逻辑等）。
