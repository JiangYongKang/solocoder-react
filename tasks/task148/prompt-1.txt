在当前项目中实现「协议条款生成器」页面，使用 localStorage 做数据持久化。

这个页面是一个协议条款文档的快速生成工具，用户可以从预设模板中选择条款类型（隐私政策、用户协议、免责声明），填写变量（公司名、日期、URL 等）后自动替换生成完整条款文档，支持 Markdown 实时预览、版本管理和导出为 HTML 文件。

1. 条款模板选择：页面左侧展示模板选择区域——以卡片列表形式展示预设条款模板。至少包含三种模板——「隐私政策」（包含信息收集、使用、存储、共享、用户权利等章节大纲）、「用户协议」（包含服务说明、用户责任、知识产权、免责、终止条款等）、「免责声明」（包含风险告知、责任限制、第三方链接、变更通知等）。每张模板卡片显示模板名称、简短描述和条款章节预览（前 3 个章节标题）。点击模板卡片选中，右侧编辑区自动加载模板内容。提供「自定义模板」——可从头创建空白模板，自定义标题和章节结构，保存后加入模板列表。
2. 变量替换：页面顶部展示变量配置面板——以表单形式展示当前模板需要的变量列表。通用变量包含「公司名称」「生效日期」「公司网址」「联系邮箱」。不同模板可能有额外专属变量——如隐私政策模板有「数据存储期限」（选项——30 天/90 天/1 年/永久），用户协议模板有「服务名称」「管辖法院」。变量输入框旁显示变量名标签（如 `{公司名称}`）。修改变量值时下方预览区实时替换更新。变量值通过 localStorage 记忆——下次打开页面时自动回填上次使用的变量值。
3. Markdown 预览：页面右侧为 Markdown 渲染预览区——模板内容以 Markdown 格式编辑（左侧编辑），实时渲染为 HTML 预览（右侧预览）。支持 Markdown 标题——一级到四级标题对应条款章节层级。支持列表——有序和无序列表用于条款细分。支持加粗和斜体——用于重要条款强调。支持链接——用于引用相关文档。预览区支持上下滚动浏览完整文档。提供「编辑/预览分栏」「纯编辑」「纯预览」三种视图模式切换。
4. 版本管理：页面底部或侧边栏展示版本历史面板——每次点击「保存版本」时自动保存当前编辑内容和变量值为一个快照版本。版本列表按时间倒序展示——显示版本号（V1, V2, V3...）、保存时间、备注（可选填写）。支持「加载版本」——点击版本恢复到编辑区。支持「版本对比」——选中两个版本后以 diff 视图展示内容变化（新增绿色、删除红色）。最多保存 20 个版本，超出时提示删除旧版本。版本数据持久化到 localStorage。
5. 导出 HTML：顶部工具栏提供「导出 HTML」按钮——将当前 Markdown 渲染后的条款文档导出为独立的 HTML 文件下载。导出 HTML 包含内联样式——标题层级、段落间距、列表缩进、加粗斜体等排版样式。HTML 文件顶部自动添加文档标题和生成日期。文档底部自动添加版权声明——"本文档由 [公司名称] 生成于 [日期]"。导出前提供预览弹窗——确认导出内容的格式效果。同时支持「复制 Markdown 源码」按钮一键复制原始 Markdown 文本。
6. 代码实现需要放在 src/pages/terms-generator/ 下，测试代码放在 src/pages/__tests__/terms-generator/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如变量替换解析算法、Markdown 到 HTML 转换的文本处理函数、版本快照的 diff 对比算法、模板数据结构校验、导出 HTML 模板字符串拼接等）。
