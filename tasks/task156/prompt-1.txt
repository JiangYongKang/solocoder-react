在当前项目中实现「像素字体生成器」页面。该页面是首页任务卡片「像素字体生成器」点击后的二级页面，使用 Canvas 绘制点阵字形。

这个页面是一个像素风格的字体设计工具，用户逐像素编辑字形并导出为可用的字体格式。

1. 点阵网格编辑：页面中央展示一个可配置大小的点阵网格（默认 16×16），每个格子代表一个像素点。用户点击格子可以点亮（填充前景色）或熄灭（变回背景色），支持按住鼠标拖拽连续点亮多个格子。网格线用浅色细线绘制，方便定位。
2. 字号与间距调节：提供字号滑块（如调节点阵网格的尺寸，12×12 到 32×32）和字符间距调节（影响预览时的文字间距）。调节字号时已编辑的点阵内容尽量保留（居中或缩放适配新网格）。
3. 文字预览：页面提供输入框，用户输入一段文本后，在预览区用当前编辑的所有字形按顺序渲染出来。预览区展示像素字体的实际效果，每个字形用当前字号渲染。
4. 多字形切换：提供字符选择面板，展示当前已编辑的字形列表（如 A-Z、0-9 等字符的缩略图）。点击某个字形的缩略图切换到该字形的网格编辑区进行编辑。支持在字符选择面板中添加新的字符（输入 Unicode 码点或直接输入字符）。
5. CSS @font-face 格式导出：将当前编辑的所有字形导出为一个 CSS @font-face 规则格式的文本。格式应包含 font-family 名称、src 使用 base64 编码的字体数据（模拟）、以及每个字符的点阵数据以某种形式嵌入。导出的 CSS 文本可一键复制。
6. JSON 保存与加载：提供「保存」和「加载」按钮，将当前所有字形数据导出为 JSON 文件下载，以及从 JSON 文件导入恢复之前的编辑状态。同时自动保存到 localStorage，刷新页面时不丢失数据。
7. 代码实现需要放在 src/pages/pixel-font/ 下，测试代码放在 src/pages/__tests__/pixel-font/ 下。还需要为该页面编写纯 JS 函数的单元测试，只测和页面渲染无关的逻辑（如点阵数据编码解码、网格缩放适配、CSS 生成等）。
